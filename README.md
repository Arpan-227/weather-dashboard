# 🌦️ Weather Dashboard

A modern and dynamic weather app built with **React**, **Tailwind CSS**, and **OpenWeatherMap API**. Supports real-time weather, forecasts, charts, and more!

## 🔧 Features

- 🔎 Search by city name
- 🔊 Voice search (Web Speech API)
- 📍 Auto detect current location
- 🌗 Dark mode / Light mode toggle
- 🌈 Dynamic weather backgrounds
- 📈 Recharts-based weather trend chart
- ⏳ 24-hour scrollable forecast
- ⚠️ Weather alerts (extreme, storms, etc.)
- 🧭 Air quality index and UV index
- 🌄 Sunrise, sunset, moon phase
- 🗺️ Map with weather overlays (Leaflet.js)
- 📱 Responsive + PWA-ready

## 🚀 Tech Stack

- **React**
- **Tailwind CSS**
- **Vite**
- **Recharts**
- **OpenWeatherMap API**
- **Geolocation API**
- **Web Speech API**
